O barramento de endereços (BE) define quantas posições de memória o processador consegue identificar. Cada endereço é um número binário — então a largura em bits determina a quantidade máxima de posições endereçáveis.

1
Entenda a lógica: bits determinam combinações
Com n fios (bits), você pode representar 2ⁿ endereços diferentes. É como perguntar: "de quantas formas posso ligar/desligar n lâmpadas?"
2 bits → 2² = 4 endereços (00, 01, 10, 11) 8 bits → 2⁸ = 256 endereços n bits → 2ⁿ endereços
2

Os prefixos quilo, mega, giga e tera são sempre potências de 2. Você vai precisar saber esses valores de cabeça.

3
Escreva tudo como potência de 2 e some os expoentes
Quando dois números com a mesma base são multiplicados, os expoentes se somam: 2ᵃ × 2ᵇ = 2^(a+b). Esse é o truque central de todo cálculo de barramento.
4 M endereços = 2² × 2²⁰ = 2²² → BE precisa de 22 bits
4
Fórmula geral para o total de pinos
Os três barramentos internos compartilham os pinos físicos do processador. Conhecendo dois, você descobre o terceiro.
Total de pinos = BD + BE + BC
